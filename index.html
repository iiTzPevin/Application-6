Heart Monitoring System

Application 6

Project Description

This project simulates a monitor system for a healthcare company in. The system uses an ESP32 microcontroller running FreeRTOS to monitor a patient's heart rate in real time using an analog sensor and allows the patient to press an emergency button to request medical attention. The system uses an ISR for immediate button response, a queue for task communication, and semaphores and a mutex for synchronization.
